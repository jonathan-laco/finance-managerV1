Configurações Gerais

Se desativado, novos usuários não poderão se registrar no sistema.
Se desativado, usuários não poderão se registrar como MEI.
Se ativado, novos usuários precisarão ser aprovados por um administrador antes de acessar o sistema.
Informações importantes:
  • Desativar o registro não afeta os usuários já cadastrados.
  • Desativar o registro MEI não afeta os usuários MEI já cadastrados.
  • Ativar a aprovação de cadastros só afeta novos registros, usuários existentes não serão afetados.
